# cloud_repository
Сетевое хранилище на java

библиотека Netty
интерфейс java FX

Для того чтобы приложение корректно работало необходимо в корне проекта создать папки client_storage и server_storage

1. загрузка на сервер пакетами 
2. скачивание с сервера
3. удаление файлов на клиенте и сервере
4. создание файла клиенте
